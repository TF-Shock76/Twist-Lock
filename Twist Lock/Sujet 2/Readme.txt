Vous trouverez dans ce dossier :
	- Le diaporama de notre soutenance (.pdf et .pptx).
	- Le projet avec les classes.

Pour lancer le jeu :
	- Sous Windows :
		- Double cliquer sur le runServer.bat (en premier)
		- Double cliquer sur le runClient.bat pour chaque client (ensuite)
		OU
		- Dans le dossier src, ouvrir un terminal et exécuter la commande : javac -encoding utf-8 *.java
		- Dans le même dossier, ouvrir un terminal et exécuter : java Server
		- Dans le même dossier, ouvrir un terminal par client et exécuter : java ClientTest

	- Sous linux :
		- Dans le dossier src, ouvrir un terminal et exécuter la commande : javac -encoding utf-8 *.java
		ET
		- Pour lancer le programme, il suffit ensuite de faire : java Server
		- puis ouvrir un terminal par client et exécuter la commande : java ClientTest
