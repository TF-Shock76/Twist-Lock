Vous trouverez dans ce dossier :
	- Le diaporama de notre soutenance (.pdf et .pptx).
	- Le projet avec 3 dossiers ayant un nom significatif:
		- Documentation
		- Les .jar
		- Les classes

Pour lancer le jeu :
	- Sous Windows et Linux:
		- Pour lancer le serveur : 
			- Dans le dossier avec les .jar, sur l'invite de commande exécutez : "java -jar GameServerUDP.jar [numPort]"

		- Pour lancer chaque client souhaité : 
			- Dans le dossier avec les .jar, sur un invite de commande exécutez : "java -jar [nom_du_fichier_client].jar"

		- Si vous souhaitez utiliser les clients : "GameClientUDP.jar", "GameClientMinimaxObf.jar" ou "GameClientGreedyObf.jar" :
			- Vous devez compléter le fichier "config.properties" avec l'adresse et le port souhaités
		
		- Suite à l'exécution des clients : "Client.jar" ou "ClientIA.jar, vous devrez entrer l'adresse et le port dans l'invite de commande.
