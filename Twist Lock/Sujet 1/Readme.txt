Vous trouverez dans ce dossier :
	- Le diaporama de notre soutenance (.pdf et .pptx)
	- Deux fichiers de tests pour les grosses méthodes
	- Le projet avec les classes.

Pour lancer le jeu :
	- Sous Windows :
		- Double cliquer sur le run.bat
		OU
		- Ouvrir un terminal et exécuter le run.bat

	- Sous linux :
		- Dans le dossier src, ouvrir un terminal et exécuter la commande : javac -encoding utf-8 *.java
		ET
		- Pour lancer le programme, il suffit ensuite de faire : java Jeu
